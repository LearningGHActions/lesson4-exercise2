SkyMaps API SDK Documentation

Version 1.0.0

Welcome

Welcome to the SkyMaps API SDK! This SDK provides simplified methods to access astronomical data and visualizations for stars, planets, and other celestial bodies.

Installation

Install using npm:

npm install skymaps-sdk

Or using pip:

pip install skymaps-sdk

Configuration

Initialize the SDK with your API key:

const SkyMaps = require('skymaps-sdk');
const client = new SkyMaps('YOUR_API_KEY');

Or in Python:

from skymaps_sdk import SkyMaps
client = SkyMaps(api_key='YOUR_API_KEY')

Basic Usage

Get Current Star Map

Retrieve a star map based on the current time and observer's location:

client.getStarMap({ latitude: 40.7128, longitude: -74.0060 }, (error, map) => { if (error) console.error(error); else console.log(map); });

Or in Python:

map = client.get_star_map(latitude=40.7128, longitude=-74.0060)
if map: print(map) else: print("Error retrieving star map")

Search Celestial Object

Search for data on a specific celestial object by name:

client.searchCelestialObject('Vega', (error, data) => { if (error) console.error(error); else console.log(data); });

Or in Python:

data = client.search_celestial_object('Vega')
if data: print(data) else: print("Error retrieving data")

Advanced Features

Customize observational parameters like time, date, and specific celestial events.

const params = { time: '2024-06-08T20:00:00', event: 'lunar_eclipse' };
client.getEventMap(params, (error, map) => { if (error) console.error(error); else console.log(map); });
